📥 HOW TO UPDATE THE DATABASE

To update, export your two sheets as CSV files named polities.csv and phases.csv and overwrite them in the same folder as this page. Then refresh the page.

  1. Keep sheet 1 as polities.csv with the columns you described (Political Entities, Start, End, … Overall).
  2. Keep sheet 2 as phases.csv with columns (Polity, Start Year, End Year, Region, Overall Score, Phase Name, Phase Start, Phase End, Phase Score, Notes).
  3. Save both CSVs and push to your GitHub Pages repo (or upload to Netlify). Refresh the page to load new data.

Note: the CSV parser reads the last calculated values, so Excel formulas are fine if you've saved the file after recalculation.
