📥 HOW TO UPDATE THE DATABASE

To update, export your two sheets as CSV files named polities.csv and phases.csv and overwrite them in the same folder as this page. Then refresh the page.

  1. Sheet 1 → polities.csv (columns: Political Entities, Start, End, … Overall)
  2. Sheet 2 → phases.csv (columns: Polity, Start Year, End Year, Region, Overall Score, Phase Name, Phase Start, Phase End, Phase Score, Notes)

Note: the CSV parser reads the last calculated values, so ensure Excel saved after calculations.
